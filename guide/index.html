# 介绍

Vue-Vben-Admin (opens new window)是一个基于Vue3.0 (opens new window)、Vite (opens new window)、 Ant-Design-Vue (opens new window)、TypeScript (opens new window)的后台解决方案，目标是为中大型项目开发,提供现成的开箱解决方案，二次封装组件、utils、hooks、动态菜单、权限校验、按钮级别权限控制等功能。项目所使用的技术都是前端较新的技术栈，可以用来作为项目的启动模版，可以帮助你快速搭建企业级中后台产品原型。也可以作为一个示例，用于学习vue3，vite，ts等等主流技术。该项目会一直持续跟进最新技术,将最新的东西应用与项目之中。

# 前言

关于组件

项目虽然二次封装了一些组件，但是可能不能满足大部分的要求。所以，如果组件不满足你的要求，完全可以不用甚至删除代码自己写。不必坚持使用项目自带的组件

# 环境准备

你的本地环境需要安装 yarn (opens new window)、node (opens new window) 和 git (opens new window)

# 模版

  • Vue-Vben-Admin (opens new window)

该版本主要是提供一些 Demo 示例及插件的使用集成方式，主要用于参考。如果对项目不是很熟悉，不建议在这基础上进行开发，可以使用下方提供的精简版本。

  • Vue-Vben-Admin-Thin-Next (opens new window)

Vue-Vben-Admin精简版本。删除了相关示例，无用文件及功能、依赖。可以根据自身要求安装对应的依赖。因为使用的是vite，依赖删除不会导致相关组件或者hook发出及警告。只在需要的时候安装对应的库即可。

# 生态

如果这些插件对你有帮助，可以给一个 star 支持下

  • vite-plugin-mock (opens new window) - 用于本地及开发环境数据 mock
  • vite-plugin-html (opens new window) - 用于 html 模版转换
  • vite-plugin-style-import (opens new window) - 用于组件库样式按需引入
  • vite-plugin-imagemin (opens new window) - 用于打包压缩图片资源
  • vite-plugin-theme (opens new window) - 用于在线切换主题色等颜色相关配置
  • vite-plugin-compression (opens new window) - 用于打包输入.gz|.br文件
  • vite-plugin-svg-icons (opens new window) - 快速生成 svg 雪碧图
